# PCCOE College Webpage - Assignment 1

A static, nicely formatted webpage showcasing the Pimpri Chinchwad College of Engineering (PCCOE) details. Built using fundamental HTML elements.

## 🚀 Features
- **College Info:** Displays basic information, vision, and mission of PCCOE.
- **Department Details:** A structured HTML table outlining the various departments, HODs, and their respective intake capacities.
- **Marquee Alerts:** Dynamic scrolling text for important announcements and news.
- **Semantic HTML:** Implementation of standard tags such as `<blockquote>`, `<q>`, `<abbr>`, `<del>`, `<ins>`, and standard lists (`<ul>`, `<li>`).

## 🛠️ Technology Stack
- **HTML5:** For the core structure and content presentation.

## 📂 Project Structure
- `index.html`: The main entry point showcasing the college webpage.

## 💡 Learning Outcomes
This assignment focuses on the fundamentals of web design using raw HTML. It demonstrates text formatting, table generation, and creating a cohesive webpage layout without the dependence on CSS or advanced frameworks.
