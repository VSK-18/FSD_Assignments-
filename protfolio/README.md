# Personal Portfolio Website

A sleek, responsive, and modern personal portfolio website showcasing Vardhan Khinvasara's projects, experience, and skills in Software Engineering and Robotics.

## 👨‍💻 About This Project
This portfolio serves as a central hub for highlighting technical proficiency, previous work, and contact information. It features a clean aesthetic with a focus on usability and modern web design principles.

## ✨ Key Features
- **Hero Section:** A captivating introduction with a professional profile image and quick summary.
- **Experience Timeline:** Detailed breakdown of current and past professional roles, including work at Team Automatons and JITO.
- **Skills Matrix:** A visual representation of technical and soft skills (Python, C++, ROS, OpenCV, etc.).
- **Project Showcase:** Highlights notable projects such as the Attendance Monitoring System, Autonomous Arrow Detection, and FTC 2025 Robot.
- **Responsive Layout:** Built with Bootstrap 5 ensuring the website looks great on all devices, from mobile to desktop.

## 🛠️ Technologies Used
- **HTML5 & CSS3:** Core structure and custom styling (`style.css`).
- **Bootstrap 5:** For rapid, responsive grid layouts and pre-styled components.
- **FontAwesome:** For intuitive, scalable vector icons.
- **Google Fonts:** Utilizing 'Fira Code' and 'Inter' for elegant typography.

## 🚀 How to View
Simply open the `index.html` file in any modern web browser to view the portfolio.
