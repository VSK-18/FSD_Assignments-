# Full Stack Development (FSD) Assignments

Welcome to the central repository for the Full Stack Development assignments and projects. This collection highlights various web development skills ranging from semantic HTML formatting to full-stack MERN/PERN application architectures.

## 📂 Repository Contents

Each directory within this repository represents a unique project or assignment designed to demonstrate specific web development skills:

### 1. `assignment1/` - PCCOE Webpage
A beautifully formatted, static HTML webpage providing information about Pimpri Chinchwad College of Engineering. Focuses heavily on raw semantic HTML, robust table structures, `<marquee>` tags, and proper document flow formatting.
- **Tech:** HTML5

### 2. `Assignment5/`
*(Currently Empty)* - Reserved for an upcoming assignment.

### 3. `climate-shield-dashboard-main/` - Climate Shield Dashboard
A responsive, high-performance climate monitoring dashboard. It visualizes critical environmental metrics natively using Recharts and interactive maps via Leaflet.
- **Tech:** React, Vite, Recharts, React-Leaflet, React Router DOM

### 4. `luxejourney-travel/` - LuxeJourney Travel Agency
A backend-focused travel agency platform using MVC architectural patterns. It uses server-side rendering to parse user views effectively while pulling dynamic package details from a database.
- **Tech:** Node.js, Express.js, MongoDB, Mongoose, EJS (Embedded JavaScript)

### 5. `protfolio/` - Vardhan Khinvasara Personal Portfolio
A sleek, modern portfolio page featuring smooth layout grids, experience timelines, project showcases, and a responsive Bootstrap 5 design. Showcases skills in robotics, embedded systems, and software engineering.
- **Tech:** HTML5, CSS3, Bootstrap 5, FontAwesome

### 6. `restaurant/` - Swaad
An immersive, culturally rich front-end experience for a pure vegetarian Indian restaurant. Features real-time cart handling, dynamic menu filtering, and custom animations.
- **Tech:** HTML5, CSS3, Vanilla JS, Bootstrap 5

## 🚀 How to Run the Projects
To run any of the front-end projects (`assignment1`, `protfolio`, `restaurant`), simply open the `index.html` file in your preferred web browser.

To run full-stack or framework-based projects (`climate-shield-dashboard-main`, `luxejourney-travel`), please navigate into their respective directories and consult the localized `README.md` instructions for dependency installation (`npm install`) and execution commands.
