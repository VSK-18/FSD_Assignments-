# 🦷 32 Smiles - Dental Clinic Booking



A comprehensive online dental clinic appointment booking system built using Node.js, Express, and MongoDB. Features a vibrant front-end styled with modern glassmorphism and a robust reservation management backend.

## ✨ Key Features
- **Effortless Booking:** Users can easily schedule appointments with specific dentists.
- **Admin Dashboard:** A streamlined interface to view and manage all scheduled clinical appointments.
- **Glassmorphism UI:** Modern, vibrant Vanilla CSS styling with frosted-glass effects for a premium feel.
- **Data Persistence:** Uses MongoDB via Mongoose with structured models for Dentists and Appointments.

## 🚀 Getting Started

1. **Navigate to the Project**:
   ```bash
   cd dental-clinic-booking
   ```
2. **Database Setup**:
   Ensure MongoDB is running locally on `localhost:27017` or provide a valid `MONGO_URI` via environment variables.
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Seed Sample Data**:
   ```bash
   node seed.js
   ```
5. **Start the Application**:
   ```bash
   npm start
   ```
6. **View Online**:
   Visit `http://localhost:3000` in your browser.
