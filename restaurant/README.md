# Swaad - Indian Pure Veg Restaurant

An elegant, fully responsive website for an authentic pure vegetarian Indian restaurant named "Swaad". It features modern animations, a dynamic menu filter, and beautiful image galleries.

## 🍽️ Features
- **Immersive Hero Section:** Engaging hero banner with floating scroll indicators and fade animations.
- **Dynamic Menu:** A categorized menu system allowing users to filter dishes by Starters, Mains, Desserts, and Drinks.
- **Cart & Ordering functionality:** Includes a sliding cart sidebar, toast notifications for adding items, and a checkout button.
- **Reservation System:** A complete reservation form validating name, email, phone, date, time, and guests.
- **Testimonials Carousel:** Highlights customer reviews using a Bootstrap slide carousel.
- **Image Gallery:** Shows aesthetic culinary images using a sleek Lightbox modal.

## 🛠️ Technology Stack
- **HTML5:** Semantic architecture.
- **CSS3:** Custom styles (`style.css`) incorporating complex animations, hover effects, and a refined color palette (Gold, Cream, Dark).
- **JavaScript (Vanilla):** Powers the cart interaction, menu filtering, reservation form handling, and the lightbox gallery (`script.js`).
- **Bootstrap 5:** Provides the responsive grid system, navbar, and carousel components.
- **FontAwesome:** Integrated icons for UI enhancements.

## 🚀 How to View
To interact with the website, open `index.html` in your browser. All interactions like the cart notification and the menu filters are handled via client-side JavaScript.
